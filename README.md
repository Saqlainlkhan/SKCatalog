# SKCatalog

A class registration web application built with ASP.NET Core MVC, Entity Framework Core, SQLite, and Bootstrap.

## Features

- View all offered classes in a responsive table
- Add, edit, and delete classes (full CRUD)
- Department dropdown with validation
- Seed data loaded on first run
- Responsive navbar (collapses on small screens)

## Tech Stack

- ASP.NET Core MVC (.NET 10)
- Entity Framework Core with SQLite
- Bootstrap 5

## Run Locally

```
dotnet run
```

Then open `https://localhost:[port]` in your browser.
